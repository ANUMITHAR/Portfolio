#  Personal Portfolio Website

**Anumitha R – Personal Portfolio**

---

##  Project Description

This is a **personal portfolio website** created using **HTML and CSS**.
It showcases my **education, skills, career objective, coding practice, achievements, and contact details** in a clean and professional layout.

This project is suitable for:

* Beginners learning web development
* Academic submission
* GitHub portfolio showcase

---

##  Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout

---

##  Project Structure

```
portfolio/
│── index.html
│── style.css
│── README.md
```

##  Features

* Professional single-page portfolio
* Clean and readable layout
* Sections for:

  * Career Objective
  * Education
  * Technical Skills
  * Projects
  * Coding Practice
  * Leadership & Soft Skills
  * Certifications
  * Achievements
  * Languages
  * Declaration
* Clickable links for:

  * LinkedIn
  * GitHub
  * LeetCode
* Responsive and beginner-friendly design
